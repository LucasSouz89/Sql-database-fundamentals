## 🇺🇸 English

This repository documents my journey in mastering Relational Databases and SQL (Structured Query Language). The exercises and architectural concepts are applied using **MySQL** and the classic **Sakila** sample database.

This study is part of my technical transition/expansion to solidify my backend skills, complementing my background in Object-Oriented Programming (C#).

### 🛠️ Tech Stack & Tools
* **RDBMS:** MySQL
* **Client / IDE:** PopSQL & Visual Studio Code
* **Concepts Applied:** Relational Data Modeling, Primary/Foreign Keys, DQL (Data Query Language), Joins, Aggregation, Subqueries, Views, User & DB Management, Python.
